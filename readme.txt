Put tags in between <head></head>

== Description ==

This plugin will let you put your own title, keywords and description meta tags for each post.

Its a great tool for SEO.

This plugin helps you to put CSS Link tags, JavaScript Script tag and Other Meta tags between HTML <head></head> from admin page.

Also put your own title, keywords and description for home page and also use home page keywords and description as default.

You will find a "ZD Header Tags" Box on right panel of Post/Page writer screen to type in you title, keywords and description for particular post/page.

= Usage =

You will find a "ZD Header Tags" Box on right panel of Post/Page writer screen to type in you title, keywords and description for particular post/page.

== Installation ==

1. Upload the full directory into your 'wp-content/plugins' directory
2. Activate the plugin at the plugin administration page
3. Add title, keywords and description into "ZD Header Tags" Box on right panel of Post/Page writer screen. Use Admin page to add other tags.
